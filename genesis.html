HTML

HTML - Hypertext Markup Language is the language of most web sites and applications.
This language acts like the building structure of the web site or web application as it acts
like a building foundation. Without the foundation, other fittings like the beauty and functionality
cannot be added to the house, in like manner, without building a foundation using markup language like
HTML there will be structure to add beauty and interactivity to the web site or web application.

HTML is made up of elements that are tagged and attributed to give a desired structure. Below is an example:

Semantic tags Non-Semantic tags

HTML Tags

In HTML tags are divided into two:

  • Paired
  • Unpaired

Paired Tags

  1. Head
  2. Body
  3. Footer
  4. Paragraph
  5. List
  6. Table
  7. Header
  8. Section
  9. Div
  10. Form etc.

Unpaired Tags

  1. Image
  2. Break
  3. Meta etc.
